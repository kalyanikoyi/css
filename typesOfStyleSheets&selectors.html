Inline styling:- we directly add style properties in the tag itself by using style attribute.It is applicable to only that particular element.

Internal styling :- we use internal styling to add styling properties to only that particular page.we use style tag in head

External styling :- External style sheet can be applied to any html file. we need to create seperate css file and we need to attach that file to html file by using link tag in head

In order to use external style sheet we have to know some selectors.

we have 6 types of selectors

1.universal selector (*) .It applies styling properties to all the elements.Here i applied font-style as italic to all the elements .

2.class selector(.).using this selector we can apply styles to particular group of elemets. Here i used class selector for all h3 tags.i changed font size to 20

3.element selector(we use tag names).we can use element selector to apply styles to particular group of same tags.Here iam taking h3 tag and applying font color to blue

4.Id selector(#).If we want to apply styling to unique tag then we use id selector.now iam adding background color to this tag

5.Attribute selector (using attributes we can add style properties)
