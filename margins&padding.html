Margins

we can add margins to the text. it takes 4 values .1st value represents top, 2nd represents right, 3rd rep bottom and 4th rep left. eg. margin:10px 20px 30px 40px. If we take 2 values 1st represents top&bottom , 2nd rep left&right. eg . margin : 10px 20px If we take 3 values 1st represents top , 2nd rep left&right ,3rd rep bottom eg. margin : 10px 20px 30px. If we take 1 value it applies to 4 sides.
